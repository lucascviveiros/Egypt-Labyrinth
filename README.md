# Egypt-Labyrinth-Unity3d

This is a first person game that simulates a Labyrinth in Egypt made with unity3d. Most assets are free!

- First Person
- Sword Attack
- Some animations and sounds
- Labyrinth make in Blender
- Free Assets used
